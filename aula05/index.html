Meus projetos

Projeto 1 - A Arte de Fritar um Ovo

Este projeto é uma página desenvolvida em HTML e CSS com o objetivo de praticar a estruturação de conteúdo e a aplicação de listas ordenadas e não ordenadas. A página apresenta uma receita de forma clara e organizada, incluindo:
- Lista não ordenada para os ingredientes e utensílios necessários.
- Lista ordenada para o passo a passo da receita.
- Estilização básica com CSS, destacando títulos e seções para melhor leitura.
- Imagem ilustrativa para complementar o conteúdo e torná-lo mais atraente.
- Esse projeto demonstra meu aprendizado nos fundamentos de HTML e CSS, com foco na organização, semântica e apresentação de conteúdo.

Projeto 2 - Clube do Filme

Este projeto é um site desenvolvido em HTML e CSS com o objetivo de praticar a criação de páginas interligadas e a organização de conteúdos multimídia. O site funciona como um espaço colaborativo fictício para amantes de cinema, onde é possível acessar informações sobre filmes e suas resenhas.

Principais características:

Página inicial: Apresenta a proposta do clube e uma galeria com pôsteres de filmes.
Página de resenha: Contém detalhes sobre cada obra, incluindo título, direção, elenco, sinopse, avaliação com estrelas e links de onde assistir.
Links de navegação: Interligam a página inicial com as páginas de resenhas.
Design simples e funcional: Cores personalizadas e layout limpo, garantindo fácil navegação.
